# Dog API App

Interactive web application that uses a public dog API to display random dog images with a clean responsive interface and dynamic content loading.

## Features

- Random dog image generation
- API data fetching
- Responsive modern UI
- Dynamic content updates
- Error handling
- Fast and lightweight interface
- Simple and user-friendly layout

## Technologies

- HTML5
- CSS3
- JavaScript
- REST API

## Functionality

- Fetches random dog images from an external API
- Updates content dynamically without page reload
- Handles loading and request errors gracefully

## Goals

This project was created to practice working with APIs, asynchronous JavaScript, and dynamic frontend interactions while maintaining a clean and responsive user interface.
